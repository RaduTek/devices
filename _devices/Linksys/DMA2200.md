## Story

I bought this item brand new in box for less than $20 delivered. It has a manufacturing date of March 2008, and I unpacked it in March 2026, 18 years later.

Setup was easy, easily connected to my modern Wi-Fi 6 network and paired without trouble to Windows Media Center running on Windows 7. The interface is a bit sluggish to use, but it may work better on a wired connection.

The DVD player works just fine, but I did get a strange prompt telling me that resolution was downgraded because of copy protection. So much for the advertised DVD upscaling, but it may work with HDMI and HDCP DRM. I've only tried with it connected via Component or SCART.

Besides DVDs, the only other thing it will play are regular audio CDs. No MP3-CDs, no data discs, nothing else. The USB port also cannot load any media.

I'm curious as to what OS this device is running. From all that I can find about the Sigma Designs SoC it seems like it only supports Linux officially, and previous players from KISS/Linksys have been confirmed to use Linux. Still, it would be really odd to see a WMC extender implemented on Linux.
